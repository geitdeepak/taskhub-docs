# Installation Guide

Get TaskHub installed and ready to use on your system.

## System Requirements

### Minimum Requirements
- **OS**: Windows 10+, macOS 10.13+, or Linux (Ubuntu 18.04+)
- **RAM**: 2GB minimum
- **Storage**: 500MB free space
- **Browser**: Chrome, Firefox, Safari, or Edge (latest versions)
- **Internet**: Active internet connection required

### Recommended Requirements
- **RAM**: 4GB or more
- **Storage**: 1GB free space
- **Processor**: Multi-core processor (Intel i5 or equivalent)

## Web Application

### Browser Installation
TaskHub works directly in your web browser with no installation required.

1. Open your web browser
2. Navigate to [https://taskhub.app](https://taskhub.app)
3. Sign up for a new account or log in
4. Start managing tasks immediately

## Desktop Application

### Windows Installation
1. Download the installer from [taskhub.app/download](https://taskhub.app/download)
2. Run `TaskHub-Installer.exe`
3. Follow the installation wizard
4. Click "Finish" to launch TaskHub

### macOS Installation
1. Download the `.dmg` file from [taskhub.app/download](https://taskhub.app/download)
2. Open the downloaded file
3. Drag TaskHub to the Applications folder
4. Launch from Applications or Spotlight

### Linux Installation

**Ubuntu/Debian:**
```bash
wget https://taskhub.app/download/taskhub-linux.deb
sudo dpkg -i taskhub-linux.deb
taskhub
```

**Fedora/RHEL:**
```bash
wget https://taskhub.app/download/taskhub-linux.rpm
sudo rpm -i taskhub-linux.rpm
taskhub
```

## Troubleshooting

### Installation Issues

**"File is corrupted" error**
- Clear your browser cache
- Download the installer again from the official website
- Temporarily disable antivirus software during installation

**"Port already in use" error**
- Close other applications using the same port
- Restart your system
- Contact support if issue persists

![logo](./images/blog91.png)

**Installation hangs**
- Check your internet connection
- Try disabling VPN temporarily
- Restart the installation process

### Common Solutions
- Ensure you have admin/sudo privileges
- Disable antivirus temporarily during installation
- Update your operating system to the latest version
- Clear temporary files and try again

## Uninstallation

### Windows
1. Go to Settings → Apps → Apps & Features
2. Find "TaskHub" in the list
3. Click and select "Uninstall"
4. Follow the uninstallation wizard

![logo](./images/blog95.png)

### macOS
1. Open Finder
2. Navigate to Applications
3. Drag TaskHub to the Trash
4. Empty Trash

### Linux
```bash
sudo apt remove taskhub  # Ubuntu/Debian
# or
sudo rpm -e taskhub     # Fedora/RHEL
```

### Mobile
- iOS: Long-press the app icon and select "Remove App"
- Android: Long-press the app icon and select "Uninstall"
